# MovieFlix - Free Netflix Style Movie Site

A Netflix clone that displays movie information using the TMDB API. No login required, completely free.

## Features
- 🎬 Netflix-style UI with movie rows
- 🔍 Browse trending, popular, top-rated, and upcoming movies
- 📱 Fully responsive design
- 🎥 Movie details modal with ratings and overview
- 🌐 Live demo available via GitHub Pages

## Setup Instructions

1. **Get a TMDB API Key** (Free)
   - Go to [TMDB](https://www.themoviedb.org/)
   - Create an account
   - Request an API key

2. **Update the API Key**
   - Open `script.js`
   - Replace `YOUR_API_KEY_HERE` with your actual API key

3. **Deploy to GitHub Pages**
   - Create a new repository
   - Upload all files
   - Enable GitHub Pages in repository settings
